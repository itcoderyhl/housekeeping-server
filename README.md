## 【原创】springboot+vue前后端分离的家政服务网系统（源码+文档）

## 系统介绍

基于SpringBoot+Vue前后端分离的家政服务网系统（源码+文档），系统包含管理员、用户角色，功能如下：
- 管理员：首页统计（评价统计、预约统计）；用户管理（用户信息、员工信息）；服务分类；服务管理；服务预约；服务评价；论坛管理（论坛版块、帖子管理、评论管理）；行业新闻；系统留言；公告信息；个人信息；密码修改。
- 员工：服务预约（完成）、服务评价（回复）、公告查看；个人信息（可上传照片和证书）、修改密码。
- 用户：注册、登录系统；首页；家政服务（服务分类、服务列表、服务详情、预约）；家政人员（查看评价、收藏）、家政论坛（版块、帖子、评论、点赞）；行业新闻；系统留言；公告信息。用户还具有个人中心功能，功能包括：服务预约、服务评价、家政收藏、我的帖子、帖子评论、帖子点赞、我的留言、个人中心、修改密码。
系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，适合作为毕业设计、课程设计，有需要可提供指导。

项目由本人原创开发，经过多次测试，项目运行无问题，请放心使用，作者QQ：1344127185。

## 使用技术

服务器端技术栈：

- Springboot
- SpringMvc
- Mybatis
- MySQL

前端技术栈：

- vue
- elementUI
- axios
- echarts

## 环境介绍

- IDEA集成环境
- JDK1.8
- MySQL5.6+

可提供所有运行软件、远程安装、技术指导。
