# housekeeping-server
基于SpringBoot+Vue前后端分离的家政服务网系统（源码+文档），系统包含管理员、用户角色，功能如下： 管理员：首页统计；用户管理（用户信息、员工信息）；服务分类；服务管理；服务预约；服务评价；论坛管理（论坛版块、帖子管理、评论管理）；行业新闻；系统留言；公告信息；个人信息；密码修改。 员工：服务预约（完成）、服务评价（回复）、公告查看；个人信息（可上传照片和证书）、修改密码。 用户：注册、登录系统；首页；家政服务（服务分类、服务列表、服务详情、预约）；家政人员（查看评价、收藏）、家政论坛（版块、帖子、评论、点赞）；行业新闻；系统留言；公告信息。用户还具有个人中心功能，功能包括：服务预约、服务评价、家政收藏、我的帖子、帖子评论、帖子点赞、我的留言、个人中心、修改密码。
